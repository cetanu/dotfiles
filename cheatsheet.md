# ⚡ Zellij Keyboard Shortcuts Cheat Sheet

Press `Esc` or `q` to exit this help screen.

---

### 🔑 Modes & Global Actions
| Action | Keybinding | Description |
| :--- | :--- | :--- |
| **Lock/Unlock** | `Ctrl + g` | Temporarily pass all keys to the active terminal/editor |
| **Quit Zellij** | `Ctrl + q` | Exit the current Zellij session |
| **Help Screen** | `Ctrl + ?` | Show this cheatsheet |
| **Floating Yazi** | `Alt + y` | Open Yazi file manager in a floating pane |

---

### 🔲 Pane Management (`Ctrl + p`)
| Action | Keybinding | Description |
| :--- | :--- | :--- |
| **New Pane** | `n` or `Alt + n` | Open a new pane in the default layout |
| **Down Split** | `d` | Split active pane horizontally |
| **Right Split** | `r` | Split active pane vertically |
| **Close Pane** | `x` | Close the current pane |
| **Toggle Floating**| `w` | Toggle whether the pane is floating |
| **Toggle Frame** | `z` | Toggle borders/headers of panes |
| **Toggle Zoom** | `f` | Toggle fullscreen zoom of the active pane |
| **Next/Prev Pane** | `Tab` / `Shift + Tab`| Cycle focus between panes |
| **Move Focus** | `Alt + h/j/k/l` | Move focus Left, Down, Up, or Right |

---

### 📑 Tab Management (`Ctrl + t`)
| Action | Keybinding | Description |
| :--- | :--- | :--- |
| **New Tab** | `n` | Create a new tab |
| **Close Tab** | `x` | Close the current tab |
| **Rename Tab** | `r` | Rename the current tab |
| **Switch Tab** | `1` - `9` | Go directly to tab index 1–9 |
| **Next/Prev Tab** | `[` / `]` or `h` / `l`| Go to previous/next tab |
| **Move Tab** | `Shift + [` / `]` | Reorder current tab left/right |

---

### ↕️ Scroll & Search Mode (`Ctrl + s`)
| Action | Keybinding | Description |
| :--- | :--- | :--- |
| **Scroll Up/Down** | `u` / `d` or `PgUp` / `PgDn` | Page up or down in history |
| **Search History**| `/` | Search text in scrollback buffer |
| **Open in Editor** | `e` | Open entire scrollback history in Neovim |

---

### ↕️ Resize (`Ctrl + n`) & Move (`Ctrl + h`)
| Action | Keybinding | Description |
| :--- | :--- | :--- |
| **Resize Direction**| `h` / `j` / `k` / `l` | Grow pane left, down, up, or right |
| **Shrink/Grow** | `-` / `+` | Decrease/increase general pane size |
| **Move Pane** | `h` / `j` / `k` / `l` (in Move mode) | Swap pane position with neighbors |

---

### 🔌 Sessions (`Ctrl + o`)
| Action | Keybinding | Description |
| :--- | :--- | :--- |
| **Detach Session** | `d` | Detach from the active session |
| **Session Manager**| `w` | Open interactive session navigator |
